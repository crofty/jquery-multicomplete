MultiComplete

Extensible auto-complete tool built on Prototype and Scriptaculous jQuery and LowPro.

Handles the basics so you can build on top of it easily instead of being stuck with whatever the author thought of.

Below are some examples of what you can do by extending the base functionality, they are all pretty simple but should show how you can build on the basics.

Going beyond these examples you could easily hook it up to search via Ajax; use different regular expressions to match with; modify the templates to display more information in the lists; use complex data types instead of simple arrays etc...

Single Selection

Text Field with inline array of data

This is setup with an inline array of data (one, two, three...ten) and is as basic as it gets

Select Box

This gets its data from the select box and when an item is selected it updates the select box.

Multiple Options

Tied to a multi-select box

This is hooked up to the select box so the suggestions will only contain those not already selected.

Advanced

Adding new items

This allows selecting multiple items, but only once per item. If what you're searching for doesn't exist, then it lets you add it.

It builds up a list of items you've selected as you go along, removing an item from this list adds it back to the list of possible suggestions.
